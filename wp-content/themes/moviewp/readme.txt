== Description ==

Write reviews, add video streaming, or create a Movie and TV Show database with MovieWP theme.

== Installation ==

1. You must do a clean installation ow WordPress before install theme and plugins!
2. If your server is PHP 8 you MUST downgrade to version 7.4 as the new version is not 100% supported by the theme.
3. Upload and install all plugins but DO NOT activate theme.
4. Upload the theme and activate it.
5. Now activate the previously loaded plugins.
6. Go to Settings> Taxonomy Image> Select the "networks" and "collection" fields.
7. Configure your setting in MovieWP admin panel.
8. Do not delete or rename category Movies and TV Series from wordpress category editor or the theme will no longer work properly!!
9. Open file license.php in theme root folder, paste license key.
