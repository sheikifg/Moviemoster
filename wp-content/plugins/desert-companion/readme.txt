== Description ==

Desert Companion Enhances <a href="https://desertthemes.com/" rel="nofollow">Desert Themes</a> with additional functionality.


**Features:**

&#9989; Import Pre-made Section in 1 Click
&#9989; Beautiful Section on Hompeage
&#9989; Manage Font Typography
&#9989; Responsive for All Devices
&#9989; Background Animations
&#9989; Translation Ready
&#9989; Edit Content Via Live Customizer
&#9989; WooCommerce Ready
&#9989; SEO Optimized
&#9989; Fast Performance
&#9989; Highly Customizable
